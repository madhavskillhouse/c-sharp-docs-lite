# Loops

C# supports all standard loops in programming world. Listed below:

1. While loop:

    ```C#
    while (a < 20) {
        Console.WriteLine("value of a: {0}", a);
        a++;
    }
    ```

2. For loop:

    ```C#
    for (int a = 10; a < 20; a = a + 1) {
        Console.WriteLine("value of a: {0}", a);
    }
    ```

3. Do..While loop:

    ```C#
    int a = 10;
            
    /* do loop execution */
    do {
        Console.WriteLine("value of a: {0}", a);
        a = a + 1;
    } 
    while (a < 20);
    ```
